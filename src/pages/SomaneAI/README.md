# 🚀 SomaneAI - Assistant IA Intelligent avec Vraies Données

## 🎉 Nouveautés Principales

Votre SomaneAI a été complètement transformé pour :
- ✅ Utiliser vos **VRAIES données** du site
- ✅ Apprendre de ce que vous entrez
- ✅ Générer des analyses basées sur VOS chiffres
- ✅ Interface type **ChatGPT** avec graphes

### 📊 Architecture Données

```
Votre site (formulaires de ventes, stocks, etc.)
           ↓
    Base de données Django
           ↓
    API REST (/api/ventes/, , etc.)
           ↓
    SomaneAI (Frontend React)
           ↓
    Graphes + Analyses en temps réel
```

### ✨ Caractéristiques Principales

#### 1. **Interface Moderne Type ChatGPT**
- Design sombre et moderne (thème Dark)
- Sidebar avec gestion des conversations multiples
- Création, suppression et renommage automatique des conversations
- Historique complet des échanges

#### 2. **Visualisation de Données avec Graphes**
Demandez à SomaneAI d'analyser vos données et il générera automatiquement des graphes :

**Types de graphes supportés :**
- 📊 **Graphiques en barres** : Comparaisons de valeurs
- 📈 **Graphiques en courbes** : Évolution temporelle
- 🥧 **Camemberts** : Répartition des parts
- 📉 **Graphiques en aires** : Tendances cumulatives

**Exemples de commandes :**
```
- "Analyse mes ventes du mois"
- "Montre-moi l'état de mon stock"
- "Rapport financier détaillé"
- "Graphique des dépenses"
- "Évolution de mon chiffre d'affaires"
```

#### 3. **Suggestions Intelligentes**
Au démarrage, vous verrez des suggestions rapides :
- Analyse mes ventes
- État de mon stock
- Rapport financier
- Optimiser mes coûts
- KPI performance
- Conseils business

#### 4. **Traitement Avancé des Requêtes**
L'IA détecte automatiquement votre sujet et répond avec des graphes pertinents :

- **Ventes** → Graphe des ventes menselles + Répartition par catégorie
- **Stock** → Évolution du stock + Articles en rupture
- **Finance** → Résultats financiers + Ventilation des dépenses
- **Performance** → KPI en temps réel

#### 5. **Actions Disponibles**
Pour chaque message de l'IA :
- 📋 **Copier** : Dupliquer le texte dans le presse-papiers
- 📊 **Graphes** : Les données sont affichées visuellement

#### 6. **Expérience Utilisateur Fluide**
- Réponses en temps réel avec animation de chargement
- Défilement automatique vers les nouveaux messages
- Support du Markdown dans les réponses
- Raccourci clavier : `Shift + Enter` pour nouvelle ligne

### 📊 Exemples de Graphes Générés

#### Analyse des Ventes

Réalisé vs Target en barres

#### Gestion de Stock

Évolution du stock en courbe

### 🎯 Comment Utiliser

1. **Posez une question** dans la barre d'entrée
2. **Consultez la réponse** avec considérations détaillées
3. **Analysez les graphes** générés automatiquement
4. **Copiez les données** si nécessaire
5. **Explorez plusieurs conversations** via la barre latérale

### 🛠️ Technologies Utilisées

- **React 18** : Framework principal
- **Recharts** : Visualisation des graphes
- **React Markdown** : Rendu des textes formatés
- **Tailwind CSS** : Stylisation complète
- **Feather Icons** : Icônes modernes

### 🎨 Design

- 🌙 **Thème sombre** pour une meilleure lisibilité
- 🎨 **Gradients** violet et rose pour l'accent
- ⚡ **Animations fluides** et microinteractions
- 📱 **Responsive** sur tous les appareils

### 💡 Conseils d'Utilisation

1. **Soyez spécifique** : "Analyse mes ventes de juin" plutôt que "ventes"
2. **Utilisez les mots-clés** : stock, vente, finance, coût, rapport
3. **Explorez les graphes** : Survolez les barres pour voir les détails
4. **Créez plusieurs chats** : Un par sujet d'analyse
